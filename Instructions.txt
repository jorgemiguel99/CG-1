1) Comandos possíveis no terminal:

	Plano: "generator plane <length> <width> plane.3d"
	Caixa: "generator box <length> <width> <height> box.3d"
	Esfera:	"generator sphere <radius> <slices> <stacks> sphere.3d"
	Cone: "generator cone <radius> <height> <slices> <stacks> cone.3d"


2) Efetuar movimentos e rotacoes:

	NOTA: Clicar primeiro no ecra se nao funcionar

	SETA DIREITA -> move para a direita
	SETA ESQUERDA -> move para a esquerda
	SETA CIMA -> move para cima
	SETA BAIXO -> move para baixo
	+ -> zoom in
	- -> zoom out
	w -> rotação em x para a frente
	s -> rotação em x para trás
	d -> rotação em z para a direita
	a -> rotação em z para a esquerda
	e -> rotação em y para a direita
	q -> rotação em y para a esquerda 

3) Ficheiros .3d gerados com:
	
	plane.3d -> generator plane 5 3 plane
	box.3d   -> generator box 5 3 10 box
